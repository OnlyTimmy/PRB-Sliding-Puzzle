# PRB – Sliding Puzzle (C# Console)

C# console application implementing a sliding puzzle game.

## Project Structure

- main → v1.0 Base assignment  
- uitbreiding → v1.1 Extended version with additional validations and features  

## Features (Base)

- Load maps from file
- Validate map dimensions
- Validate allowed characters
- Ensure exactly one player
- Ensure at least one ice tile
- Console-based board rendering
- Basic movement system

## Features (Uitbreiding)

- Extra map validation rules
- Isolated ice detection
- Extended error handling
- Improved structure and logic separation
- Cleaner validation flow

## Technical Details

- Language: C#
- Type: .NET Console Application
- OOP principles applied
- Input validation & structured error handling
- File-based map loading

## How to Run

1. Open the solution file:
   PRB99.ASN.SlidingPuzzle.Tim_Harmsen.sln

2. Or run via CLI:
   dotnet run

## Versioning

- v1.0-base → Original assignment
- v1.1 → Extension with advanced validation
